Placement Test

The purpose behind developing this project is to automate the technical round of the campus hiring process where recruiters can conduct MCQ-based technical tests. It will automatically identify whether the answer selected by the user is correct or not. It will count the obtained marks of the user and notify the user regarding the next round through email.

If any suspicious activity is detected from the user side like try to change the web page using inspect or try to modify the timer or pressing any irrelevant keys during exam then the user will be disqualified immediately. Users can continue the exam even after restarting the pc but time deduction will continue throughout the process.
